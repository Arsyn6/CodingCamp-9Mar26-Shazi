# Product Overview

## Productivity Dashboard

A client-side web application providing essential productivity tools in a single interface. Users can manage time and tasks without backend infrastructure or installation.

## Core Features

- Real-time clock with personalized greetings
- 25-minute focus timer for time-boxed work sessions
- Task management with sorting and optional due dates
- Quick links to favorite websites
- Light/dark theme support
- Local browser storage for data persistence

## Key Characteristics

- Zero configuration - works immediately by opening HTML file
- No server required - runs entirely in browser
- No network requests during normal operation
- Cross-browser compatible (Chrome, Firefox, Edge, Safari)
- Can be deployed as standalone HTML or browser extension
